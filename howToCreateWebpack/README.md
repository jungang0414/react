# Webpack 

使用Webpack來做React設定

#### 在專案目錄底下
```
//初始化
npm init -y
```

#### 安裝 webpack 工具
```
npm i webpack webpack-cli webpack-dev-server --save-dev
```

#### 安裝 babel 工具 (*.js檔案)
```
npm i babel-loader @babel/core @babel/preset-env @babel/runtime @babel/plugin-transform-runtime @babel/preset-react --save-dev
```
